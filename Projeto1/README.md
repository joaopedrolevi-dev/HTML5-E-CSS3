# site-html-css, projeto1

## Sobre o Projeto

Projeto desenvolvido como parte dos meus estudos em **HTML** e **CSS**, com foco na aplicação de cores, cores de fundo e estilização de elementos visuais.

O principal objetivo foi praticar a utilização de **CSS externo**, adotando uma estrutura organizada e alinhada às boas práticas de desenvolvimento front-end. Apesar de ser um projeto simples, ele foi importante para consolidar conceitos fundamentais e fortalecer minha base em desenvolvimento web.

Atualmente, sou estudante de **Engenharia de Software** e estou em preparação para iniciar minha trajetória profissional por meio de um estágio. Este projeto representa um dos meus primeiros passos na construção de conhecimento prático e do meu portfólio como desenvolvedor.
